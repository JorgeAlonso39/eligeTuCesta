# eligeTuCesta
Esta aplicación Java gestiona datos de personal mediante un flujo completo de ficheros. Transforma datos en bruto desde TXT a archivos binarios .dat (serialización de objetos), para luego procesarlos y generar informes XML categorizados por edad (menores de 25 y mayores de 55). Incluye validación de datos corruptos y manejo robusto de excepciones.
